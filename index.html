Template Notes

Converted concepts

Latitude estimator, landmark distance calculations, approximate MGRS-style formatting, terrain scoring, scenario risk review, resource tracker, and report export.

Data storage

This template uses browser localStorage. No server is required. Open index.html directly or host the folder on any static web server.

Accuracy

The original Python files used placeholder logic in several places. This HTML version keeps those ideas as training/planning aids and labels approximate calculations clearly.
